# Asteroids
Simple remake of the classic 1979 Asteroids arcade game. Started using the guided project from boot.dev, I intend to add more functionality and features.

## Planned Additions
### Short(ish) term
- Sound (Wow!)
- Colors
- Score boosting drops
- Boost!
- Asteroid break effects
- Asteroid collision
- More variance in asteroid speeds
- Different weapons
- Thruster trails

### Long term
- Transition to a more robust language for extra features
- A proper GUI
- Enemy ships
